## Demo Flow Sequence:

1. **Organization Setup** (2 min) → Create 2-3 organizations
2. **Entity Creation** (3 min) → Add 3-4 entities under organizations
3. **Client Onboarding** (5 min) → Create 3 clients with different linking patterns
4. **Client Acceptance** (3 min) → Complete acceptance for 2 clients
5. **Conflict Check** (2 min) → Perform checks
6. **Engagement Letter** (3 min) → Create and sign letter
7. **Engagement Creation** (5 min) → Template-based engagement setup
8. **Engagement Planning** (5 min) → Budget, milestones, time tracking
9. **Trial Balance** (5 min) → Import CSV, validate, map accounts
10. **Adjustments** (2 min) → Create AJE
11. **Financial Statements** (3 min) → Generate EAS statements
12. **Working Papers** (3 min) → Create and review WPs
13. **Activity Log** (2 min) → Show blockchain verification and revert

**Total Demo Time: 40-45 minutes**
